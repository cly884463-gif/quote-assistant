# 内部体验版上线清单

## 推荐方式

当前阶段建议先上传为微信小程序体验版，供公司内部固定成员使用。

## 上传前检查

1. 使用微信开发者工具打开项目目录：`F:\Buyto SML`
2. 确认 AppID：`wx805ee94eabb4e25a`
3. 确认首页、创建经销商批发报价、创建渠道合作报价、生成清单、导出报价图片都能正常操作。
4. 确认微信公众平台后台已添加内部 12 人为体验成员。
5. 确认小程序后台隐私保护指引已覆盖“保存图片到相册”相关能力。

## 上传体验版

1. 微信开发者工具右上角点击“上传”。
2. 版本号建议：`0.1.0`
3. 项目备注建议：`内部报价工具体验版：经销商报价、渠道合作报价、色浆产品、报价图片导出`
4. 上传后进入微信公众平台小程序后台。
5. 在“版本管理”中设为体验版。
6. 复制体验二维码发给内部成员。

## 正式发布前建议

1. 加内部权限控制，只允许指定人员使用。
2. 后台化报价表导入，避免每次价格变更都重新上传代码。
3. 正式完善隐私协议和用户说明。
4. 用真实业务订单做 3-5 次完整报价测试。
