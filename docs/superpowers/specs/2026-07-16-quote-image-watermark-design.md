# 报价图片水印设计

## 目标

在经销商批发报价和渠道合作报价导出的最终 PNG 图片中加入浅色品牌水印，同时保持表格、金额、备注和注意事项清晰可读。

## 展示规则

- 水印文字固定为 `新材联·New Material Union`。
- 水印仅出现在导出的报价图片及其预览中，不显示在网页报价清单页面。
- 水印以约 `-24°` 斜向重复排列，横纵间距固定，覆盖完整画布。
- 水印使用浅灰色，画布透明度设为 `0.06`，比初版 `0.045` 略微加深。
- 水印绘制在白色背景之后、表格边框和所有报价文字之前，保证正文始终位于水印上层。

## 实现方式

在 `web/app.js` 中增加独立的 `drawQuoteWatermark(ctx, width, height)` 绘制函数。函数通过 Canvas 的 `save`、`rotate`、`fillText` 和 `restore` 完成重复水印，并在 `exportQuoteImage()` 完成白色背景填充后立即调用。

绘制函数必须恢复 Canvas 状态，避免透明度、旋转角度、字体和文字对齐方式影响后续报价表格绘制。

## 测试与验收

- 静态测试确认水印文字、绘制函数、透明度和导出流程调用均存在。
- 运行全部现有测试，保证报价数据、清单、导出和部署配置没有回退。
- 重新生成 Netlify 发布包，确认构建成功。
- 验收标准：导出图片全页可见浅灰斜向水印，表格内容不被遮挡，经销商与渠道报价表现一致。
