# 陶釉特调色浆按颜色选择调色费设计

## 目标

将陶釉特调色浆当前自动附加 50 元调色费的行为改为用户主动选择。每一种参考颜色视为一项独立需求，同一种颜色只收取一次调色费，不同颜色分别计费。

## 卡片交互

- 陶釉特调色浆卡片增加复选框：`需要调色（+¥50/颜色）`。
- 复选框默认不勾选。
- 用户填写规格、单价、参考颜色及色号、数量，并按需勾选后添加。
- 已添加状态按“产品编号 + 规格 + 参考颜色及色号”识别。
- 修改参考颜色及色号后，当前卡片恢复为白色未添加状态，已加入清单的旧颜色记录保持不变。
- 输入与已添加记录相同的规格和参考颜色时，卡片恢复该记录的数量、价格及勾选状态，允许更新或删除。

## 清单与费用

- 每条陶釉特调色浆记录保存 `needsTintingFee` 勾选状态。
- 对所有已添加记录按去除首尾空格后的参考颜色及色号分组。
- 同一种颜色只要至少一条记录勾选“需要调色”，清单生成一条 50 元调色费。
- 同一种颜色存在多个规格时仍只生成一条调色费。
- 不同颜色分别生成调色费，每种颜色 50 元。
- 调色费行注明对应的参考颜色及色号，便于核对。
- 当某颜色所有记录均未勾选，或该颜色的最后一条记录被删除时，对应调色费自动移除。

## 数据与兼容

- 普通产品仍沿用“产品编号 + 规格”的清单键，不受影响。
- 陶釉特调色浆使用“产品编号 + 规格 + 参考颜色及色号”的清单键。
- 费用记录使用稳定的颜色键，避免不同颜色之间互相覆盖。
- 首页材料预览不增加复选框，仅创建报价页面展示该控件。
- 报价页面、最终清单、报价图片和受保护 Excel 导出都读取同一份清单数据，因此同步显示按颜色生成的调色费。

## 验证

- 未勾选时不生成调色费。
- 单色勾选时只生成一条 50 元调色费。
- 同色多规格勾选时仍只生成一条调色费。
- 多色勾选时每种颜色各生成一条调色费。
- 修改颜色后旧记录保留，卡片进入新的未添加状态。
- 删除或取消某颜色的最后一个计费需求后，对应调色费消失。
- 现有报价图片、Excel 导出和其他产品选择测试继续通过。
