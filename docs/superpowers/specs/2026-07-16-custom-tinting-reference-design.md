# 特调色浆参考颜色及色号设计

## 目标

在陶釉特调色浆的报价编辑卡片中增加“参考颜色及色号”输入框。填写内容随产品规格保存，修改后实时同步到已添加产品，并出现在最终报价清单和导出图片中。

## 编辑行为

- 输入框只在 `TY-TT-SJ` 陶釉特调色浆卡片中显示，位于自定义单价下方、数量控件上方。
- 标签和占位文字均为“参考颜色及色号”，允许填写任意文本，例如“立邦 NN3401-4 灰咖色”。
- 字段可留空，不作为添加产品的必填条件。
- 字段按所选规格保存；2.4KG配套、5KG配套和18KG配套可以分别填写不同内容。
- 已添加产品修改该字段时，清单数据实时更新，不要求退出或重新添加。
- 切换到已添加的规格时，输入框恢复该规格已保存的内容；未添加规格显示为空。

## 清单与导出

报价条目的内部规格和唯一键保持不变，新增独立字段 `referenceColor` 保存输入内容，防止修改颜色后产生重复清单项。

生成最终清单和导出图片时：

- 有内容：产品名称显示为“陶釉特调色浆（参考颜色及色号：填写内容）”。
- 无内容：产品名称仍显示“陶釉特调色浆”。
- 自动添加的“特调调色费”不显示参考颜色。

## 验收标准

- 陶釉特调色浆卡片出现可编辑的“参考颜色及色号”输入框。
- 输入内容能够随当前规格加入清单，并在修改后实时更新。
- 三种规格分别保存各自的参考颜色及色号。
- 最终网页清单和导出图片的产品名称列显示对应内容。
- 修改参考颜色不会新增重复产品，也不会影响自定义单价、数量或自动调色费。
- 自动化测试覆盖字段保存、更新、按规格隔离及清单/导出展示。
